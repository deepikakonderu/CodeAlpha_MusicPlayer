# 🎵 CodeAlpha Music Player

A modern and responsive Music Player built using HTML, CSS, and JavaScript as part of the CodeAlpha Internship Program.

---

## 🚀 Features

- 🎶 Play / Pause Music
- ⏭️ Next & Previous Controls
- 📂 Folder-Based Music Loading
- 🖼️ Automatic Album Cover Detection
- 📜 Dynamic Playlist
- ⏱️ Music Progress Bar
- 🔊 Volume Control
- 🔁 Autoplay Feature
- 💎 Modern Glassmorphism UI
- 📱 Responsive Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
